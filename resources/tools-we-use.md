# Tools & Services Stack

## Development
- **GitHub:** Code repository + CI/CD (Actions)
- **VS Code:** IDE with Copilot
- **Cursor:** AI-powered code editor
- **Linear:** Issue tracking + sprint planning
- **Notion:** Documentation + knowledge base

## Frontend
- **Next.js 14+:** React framework
- **TypeScript:** Type safety
- **Tailwind CSS:** Styling
- **shadcn/ui:** Component library
- **Wagmi + RainbowKit:** Web3 wallet connection
- **PostHog:** Product analytics
- **Vercel:** Hosting + deployment

## Backend
- **NestJS:** Node.js framework
- **PostgreSQL 15:** Primary database (Supabase)
- **Redis 7:** Caching (Upstash)
- **BullMQ:** Job queue
- **Railway / Fly.io:** Hosting
- **Sentry:** Error tracking

## Blockchain
- **zkSync Era:** L2 for proof anchoring
- **Biconomy:** ERC-4337 account abstraction SDK
- **Hardhat:** Smart contract development
- **Ethers.js / Viem:** Blockchain interactions
- **IPFS (Filebase):** Decentralized file storage

## AI/ML
- **PyTorch:** RL model training
- **TorchScript:** Model export for production
- **FastAPI:** ML inference microservice
- **Polygon.io:** Market data feed
- **Kraken / Coinbase WebSocket:** Crypto data

## zk-Proofs
- **Circom 2:** Circuit development
- **SnarkJS:** Proof generation (WASM)
- **Groth16:** Proof system
- **Rust (Lagrange):** High-performance proof workers

## Integration
- **SnapTrade:** Unified broker API
- **Resend / SendGrid:** Transactional email
- **Stripe:** Billing (Phase 2)

## Monitoring & Observability
- **Grafana Cloud:** Metrics + dashboards
- **Prometheus:** Time-series metrics
- **BetterStack:** Uptime monitoring
- **PostHog:** User analytics + session replay

## Communication
- **Slack:** Team chat
- **Zoom:** Video calls
- **Loom:** Async video updates
- **Intercom:** User support (Phase 2)

## Security
- **1Password:** Team password manager
- **Cloudflare:** DDoS protection + CDN
- **AWS Secrets Manager:** API keys storage

## Design
- **Figma:** UI/UX design + prototyping
- **Excalidraw:** Whiteboarding
- **Miro:** Collaborative planning

## Legal & Compliance
- **DocuSign:** Contract signing
- **Deel:** Contractor payments
- **Stripe Atlas:** Company incorporation (if USA)
